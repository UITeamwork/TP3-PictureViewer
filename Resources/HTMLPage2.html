Photo Manager application
SlideShow help
Command
  • F1 : Opens the help for the slide show
  • F11 : Toggle to fullscreen and back.
  • P : Stops the timer and stay on the current picture.
  • S : Stops the timer to stay on the current image.
  • R : Toggle to Random order and back.
    #Note the order is not set on random by default.
  • Down : lower the time interval on the slideshow
  • Up : Speeds the time interval on the slideshow
  • Right : Skip to next image.
  • left : Show previous image.
  • Escape & Alt+F4:Closes the slideshow.
