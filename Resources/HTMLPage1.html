Photo Manager application
Menu help
  • Filters
    • By Users :
      Let you filter the pictures you see by user. For exemple you can choose to see only you photos,the photos of all users that ar not blacklisted or the photos of one specific user.
    • By Keyword :
      Lets you filter the photos you see by a keuword search. For example, if you search "dog", you will see all images that have "dog" in their keywords.
    • By Date interval :
      Lets you filter the photos you see by the date they were published. For example, if you search a photo published on the 2018-11-04, you would set date picker to the day before and day after.
      #note: it doesn't work if you put both date picker to the same date, because the hour is set automaticly to midnight.

    #note : you can use more than one filter at a time.

  • Manager (blacklist and SlideShow)
    • BlackList Manager : Opens the black list manager where you can manage the user list and the black list.
    • SlideShow :Opens the slideshop player that lets you see the pictures you added to the slide show and control the slideshow.
